Project Title Project brief selector is used to display Material icons in Angular.We have around 900+ Angular Material icons.To show the below list icons,We need to load material icons css provided by Google is part of angular material module called MatIconModule.We can use font ligature as an icon by putting the ligature text in component. For example home displays home icon. We have to import MatIconModule from Angular Material Modules.
